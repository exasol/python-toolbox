Styleguide
==========

TL;DR
______
Read PEP8 and the `Google Styleguide`_ for a general orientation. Most rules should be check by our tooling.

Style
_____
.. note::

    This guide is a work in progress, so if you see something which isn't addressed or should be changed
    feel free to crate a Issue/PR to start a discussion on the topic.

    .. attention::

        Please search the Issues & PR's beforehand, to make sure that your case
        is bringing something new to the table.

How to apply style & coding conventions
---------------------------------------

#. Apply formatters and linters
#. Check if we have defined an explicit rule on this in the styleguide
#. Consult the `Google Styleguide`_
#. Consult `PEP 8`_
